# Onboarding assets

Placeholder SVG illustrations used by `TourIllustration` in hero steps. Each tab's flagship Full tour hero references one of these files.

Naming: `{topic}-hero.svg`. Replace any of these with Lottie JSON (same filename, `.json`) and swap `kind: "lottie"` in the tour step — the `TourIllustration` primitive supports both.

Uses CSS variables so they track the app's theme. Safe to replace with designed artwork later.
