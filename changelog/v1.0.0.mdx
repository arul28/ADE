Initial public release.

ADE (Agentic Development Environment) is a local-first desktop application for AI-assisted software development. It provides an integrated workspace with autonomous coding agents, a virtual CTO advisor, project-aware automations, terminal lane management, and deep integrations with GitHub and Linear — all running privately on your machine.
